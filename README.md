# Personal Portfolio

## Project Overview

This project involves creating a multi-page Personal Portfolio Website using only HTML, CSS, and JavaScript. The primary objective is to showcase personal information, projects, articles, experience, and contact details in a structured and professional way.

The project focuses on building a semantic and accessible website while applying essential frontend practices such as SEO meta tags, Open Graph (OG) integration for social sharing, favicon implementation, responsive layout preparation, and clean navigation across multiple pages.

## Project Requirements

* **Semantic HTML**
  Use semantic HTML elements to structure content sections such as Homepage, Projects, Articles, Education, Experience, and Contact.

* **Search Engine Optimization (SEO) Meta Tags**
  Include essential metadata inside the head section to improve discoverability and search engine indexing.

* **Open Graph (OG) Tags**
  Implement Open Graph tags to provide optimized previews when sharing links on social media platforms.

* **Favicon**
  Add a favicon for better branding and browser recognition.

* **Multiple Pages Navigation**
  Create multiple connected pages with a consistent navigation experience.

* **Responsive Layout Preparation**
  Organize the project structure to support future responsive enhancement and scalability.

* **Interactive Elements**
  Include JavaScript functionality where needed to improve user interaction.

## Project Structure

* **Homepage (`index.html`)**
  Introduces the portfolio owner, featured projects, experience, education, and testimonials.

* **Projects Page (`projects.html`)**
  Displays portfolio projects with descriptions and showcases development work.

* **Articles Page (`articles.html`)**
  Contains article previews and learning-related content.

* **Contact Page (`contact.html`)**
  Provides a contact form for user interaction.

## How to Use

1. Clone or download this repository.

```bash
git clone https://github.com/aditnurdiansyah/personal-portfolio
```

2. Navigate to the project folder.

```bash
cd personal-portfolio
```

3. Open `index.html` in your browser.

4. Customize content, styling, and assets based on your own portfolio needs.
