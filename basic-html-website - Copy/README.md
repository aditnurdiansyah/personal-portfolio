# Basic HTML Website

## Project Overview

This project involves creating a multi-page Basic HTML Website using only HTML. The primary objective is to learn how to create multiple pages on a website, structure a website semantically using HTML, different sections of a website like header, footer, navigation, main content etc, and prepare it for future styling. The project covers essential aspects such as semantic HTML, SEO meta tags, Open Graph (OG) tags for social media sharing, inclusion of a favicon, and add a logo to the navbar.

## Project Requirements

- **Semantic HTML**  
   Use appropriate HTML tags to structure your Basic HTML Website, ensuring that each section (Homepage, Projects, Articles, Contact etc.) is well-organized and semantically correct.

- **Search Engine Optimization (SEO) Meta Tags**  
   Include essential meta tags in the head section for better search engine optimization (SEO).

- **Open Graph (OG) Tags**  
   Add OG tags in the head section to enhance social media sharing, making your Basic HTML Website more shareable and visually appealing on platforms like LinkedIn and Facebook.

- **Favicon**  
   Add a favicon to your CV page for a polished and professional look.

- **Multiple Pages**  
   The entire Basic HTML Website should be presented on a multiple pages with clearly defined sections with a navigation bar.

- **Contact Page**
    The contact page should have a form with fields like name, email, message etc.

## Project Structure

- **HTML Structure**  
  The Basic HTML Website is divided into sections including Homepage, Projects, Articles, Contact etc. Each section is marked up using semantic HTML tags for clarity and structure.

- **Head Section**  
  The head section includes essential meta tags for SEO, OG tags for social media sharing, link to the favicon.

- **Body Section**  
  The body section contains the main content of the Basic HTML Website, laid out in a clear and logical manner.

## Submission Checklist

- [x] SEO meta tags included in the head section.
- [x] OG tags for better social media sharing.
- [x] Favicon linked in the head section.

## How to Use

1. Clone or download the project files.
2. Open the `index.html` file in your web browser to view the Basic HTML Website.
3. Modify the HTML content as needed to customize the Basic HTML Website with your personal information.

## Conclusion

This project provides a solid understanding of how to create a multiple pages Basic HTML Website using HTML, apply basic SEO principles, and prepare the webpage for future styling. By completing this project, you are well-prepared to move on to more advanced web development topics.
